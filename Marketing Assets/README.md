# Kilig — Meta Ad Creative Pack (20 ads)

20 ad creatives for the Kilig demand-validation campaign. All **9:16 vertical** (Reels / Stories / TikTok / FB feed), brand-locked to the Kilig reference images in `../Reference Images/`.

- **Images:** `gpt-image-2` @ 2K, high quality (1520×2688)
- **Videos:** `seedance-2.0` @ 1080p, drama genre, ~5–6s, silent (add your own sound/caption in-platform — Meta autoplays muted)
- Every asset used a Kilig reference image so the cast, palette (Kilig Rose #FF3D68, Mango Gold #FFB627, midnight plum), and `kilig` wordmark stay consistent.
- Each maps to a best-fit **Higgsfield Marketing Studio** ad template.

> Note: videos are **silent and text-free** on purpose — overlay your hook copy + captions + a CTA end-card in Meta/CapCut per placement. Images already have baked-in Taglish hooks + CTAs.
